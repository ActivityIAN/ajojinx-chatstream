# AjoJinx ChatStream

AjoJinx ChatStream is an OBS plugin for beginner streamers who want to display YouTube live chat as a private, click-through overlay while streaming.

This repository is used as the public update channel for AjoJinx ChatStream.

## Current Version

`0.1.0`

## Update Metadata

The plugin reads update information from:

```text
https://raw.githubusercontent.com/ActivityIAN/ajojinx-chatstream/main/update.json
```
