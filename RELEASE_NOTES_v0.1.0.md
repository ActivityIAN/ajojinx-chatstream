# AjoJinx ChatStream v0.1.0

Initial testing build.

## Highlights

- Native OBS plugin dock.
- Private YouTube live chat overlay.
- Click-through overlay for single-monitor streamers.
- Basic themes and layout options.
- Installer with uninstall support.
